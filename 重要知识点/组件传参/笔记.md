### 组件之间的传参
#### 父传子
+ 父传
   - 通过自定义属性将参数交给子组件
+ 子接
   - 通过 props 属性接收父元素通过自定义定义属性传递过来的参数

#### 子传父
+ 子传
   - 子组件定义一个自定义事件,将参数交给事件
+ 父接
   - 父元素在使用子组件时给这个子组件绑定它的自定义事件,从而得到参数
   ```
     <son @事件名="方法" />
     methods: {
         方法名(参数) {
             // 接收参数
         }
     }
   ```
